Transaction Fees Explained

Bitcoin fees vary based on network congestion:

  • Medium congestion: $2 - $5 (30-60 min)

Provably Fair Gaming

What is Provably Fair?

Provably fair uses cryptographic hashing to prove game outcomes weren't manipulated. You can mathematically verify each result.

How It Works:
1. Casino generates random "server seed" (hashed, shown to player)
2. Player provides "client seed" (or auto-generated)
3. Seeds combine to create game outcome
4. After game, casino reveals server seed
5. Player verifies hash matches original

Result: Casino cannot manipulate outcome after showing hashed seed

Verifying Provably Fair Results

Example: Dice Game

  1. Before bet: Casino shows hashed server seed
  2. You set client seed: "player12345"
  3. Combined hash generates result: 47.32
  4. After bet: Casino reveals unhashed server seed
  5. You hash it yourself, compare to original
  6. If match: Game was fair
Pro Tip: Most players never verify provably fair games. Use third-party verifiers (like hashverify.com) to check periodically. Change your client seed regularly to ensure randomness.
